KitKare
</h3>
<div>
A mobile application for remote cat caring, written for <b>Android</b> (team project).
</div> 
<div>
The application sends requests via <b>ASP.NET Web API</b> to a feeding station, which pours a certain amount of food into a bowl. 
There's also a functionality for switching lights on and off. The idea includes adding water section as well as video streaming. 
The hardware part runs on an Arduino.
</div>
